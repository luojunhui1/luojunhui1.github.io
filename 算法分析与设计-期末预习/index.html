背景知识

算法的时间复杂性和空间复杂性。

  • 时间复杂性：指执行算法所消耗或占用时间资源的量
  • 空间复杂性：指执行算法所消耗或占用空间资源的量

渐近复杂性定义：对于T(n)，如果存在f(n)，使得当n→∞时有(T(n)-f(n))/T(n)→0，那么，就说f(n)是T(n)当n→∞时的渐近性态。又称算法A的渐近复杂性。

  • 大O表示法(算法运行时间的上界 )

    若存在两个正的常数c和n0，对于任意n≥n0，都有T(n)≤c×f(n)，则称f(n)是T(n)的上界，记为：T(n)=O(f(n))

  • Ω表示法 (算法运行时间的下界 )

    若存在两个正的常数c和n0，对于任意n≥n0，都有T(n)≥c×g(n)， ，则称g(n)是T(n)的下界，记为：T(n)= Ω(g(n))

  • Θ表示法 (紧渐进界，同阶符号)

    定义2.3 若存在三个正的常数c1、c2和n0，对于任意n≥n0都有c1×f(n)≥T(n)≥c2×f(n)，则称f(n)与T(n)同阶，记为：T(n)=Θ(f(n))

蛮力法

算法思想：

​ 根据问题的定义采用直接的方法进行求解。

特点：

​ 这种方法简单直接，能够最广泛地解决问题，同时也是衡量其他算法的最低标准，效率不高。

分治法

分治法是指将一个更复杂的问题分成两个或更多相同或相似的子问题，再把子问题分成更小的子问题，一直这样循环下去，直到最后子问题可以直接求解，原问题的解即子问题解的合并。

分治法能解决的问题的基本特征：

  • 该问题缩小到一定程度就能解决
  • 该问题可以分解为若干个规模较小的相同问题
  • 利用该问题分解出的子问题可以合并为该问题的解
  • 该问题所分解的各个子问题是相互独立的

分治法在每一层递归上都有三个步骤：

分治法步骤（每一层递归）
分解
解决
合并

分治法在算法上的应用：

排序算法：

  • 合并算法
  • 快速排序算法

查找问题：

  • 折半查找（有序顺序表，链表不可）
  • 二叉树遍历

数值计算问题：

  • 大整数乘法
  • Strassen 矩阵乘法

几何问题：

  • 最近对问题
  • 凸包问题
